## Proyecto de Clasificación Binaria – MLOps

### Descripción del Proyecto

El presente proyecto fue desarrollado en el marco del curso de **MLOps**, con el objetivo de implementar un **pipeline completo de entrenamiento y despliegue continuo** para un modelo de **clasificación binaria**, siguiendo buenas prácticas de ingeniería de Machine Learning y operaciones.

El trabajo abarca desde el preprocesamiento de datos y el entrenamiento del modelo hasta su contenedorización y despliegue en la nube, integrando herramientas y conceptos fundamentales de MLOps.

### Objetivos del Proyecto

- Implementar un pipeline automatizado de **preprocesamiento de datos y entrenamiento del modelo**.
- Crear una **imagen Docker** para el despliegue del modelo entrenado.
- Experimentar el **despliegue del modelo en AWS SageMaker**, evaluando un entorno de producción en la nube.

### Qué sigue

Como trabajo futuro, se consideran las siguientes extensiones del proyecto:

- Implementar mecanismos de **detección y monitoreo de data drift**, con el fin de identificar cambios en la distribución de los datos que puedan afectar el rendimiento del modelo en producción.
- Incorporar un pipeline de **CI/CD (Continuous Integration / Continuous Deployment)** para automatizar el versionado, pruebas, entrenamiento, validación y despliegue del modelo de manera continua.
